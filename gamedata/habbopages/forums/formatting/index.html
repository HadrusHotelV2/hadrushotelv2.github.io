h1. Formato de foro

Formato básico

Use * bold * para el texto bold , _italic_ - para el texto italic .



Comience una línea con el símbolo > para citar, como:
> esta es una cita

Referencia

Para citar otro Habbo, simplemente use el formato '@Habboname'. Esto se convierte en un hipervínculo dirigido al perfil de Habbo, y el usuario recibirá una notificación (aún en la fase de prueba).

Responder

Puede responder a mensajes individuales haciendo clic en el verde "Responder" desde la barra de mensajes. También puede responder a varios mensajes al mismo tiempo, dejando la ventana "Escribir mensaje" abrir haciendo clic en más botones " Responder " en la página principal del foro.
